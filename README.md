## <a name="library"></a> การติดตั้งไลบรารี่
การทำการ import ไฟล์ข้างต้นจำเป็นต้อดาวโหลดไลบรารี่ซึ่งสามารถดาวโหลดได้ด้วยการเลือกที่ เครื่องมายสามขีด > Manage pelette > Install > ทำการค้นหา > install
